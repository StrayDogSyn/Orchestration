# Module 05: Deployment

**Duration**: Weeks 17-20 (approximately 4 weeks)

## Overview

This module teaches students how to deploy and maintain AI-powered applications using free-tier cloud platforms. Focus on reliability, monitoring, and cost optimization.

## Learning Objectives

By the end of this module, students will be able to:
- Deploy applications to cloud platforms
- Set up CI/CD pipelines
- Monitor application performance
- Debug production issues
- Optimize costs within free-tier limits
- Implement basic security practices

## Module Structure

### Lectures
Core instructional content covering:
- Cloud platform options (Vercel, Railway, Render, etc.)
- Deployment strategies
- CI/CD with GitHub Actions
- Environment variables and secrets management
- Monitoring and logging
- Performance optimization
- Security best practices

### Labs
Hands-on exercises including:
- Platform setup and configuration
- Deployment practice
- CI/CD pipeline creation
- Monitoring setup
- Debugging exercises

### Projects
Portfolio-building activities:
- Deployed production application
- CI/CD pipeline implementation
- Monitoring dashboard setup

### Resources
- Cloud platform documentation
- Deployment guides
- Security checklists
- Cost optimization tips

## Assessment

Portfolio components:
- Live, publicly accessible application
- Documented deployment process
- Monitoring and maintenance plan
