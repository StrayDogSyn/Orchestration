### Module 01: Foundations (Weeks 1-3)
**Theme**: Why fundamentals matter MORE with AI

| Topic | Description |
|-------|-------------|
| Chinese Mail Room | Why you can't outsource understanding |
| Big O for Models | Time complexity as model selection |
| Data Structures | Prompt patterns from CS fundamentals |

**Deliverable**: Code Explainer that demonstrates understanding of prompt-as-function

---

### Module 02: Prompt Engineering (Weeks 4-6)
**Theme**: Prompt engineering as a programming discipline

| Topic | Description |
|-------|-------------|
| Prompt as Function | Inputs, outputs, error handling |
| Chain-of-Thought | Explicit algorithm reasoning |
| Iteration Cycles | Versioning and refinement |

**Deliverable**: Robust prompt library with documented design patterns

---

### Module 03: Orchestration (Weeks 7-9)
**Theme**: Model selection, load balancing, cost optimization

| Topic | Description |
|-------|-------------|
| Decision Trees | Which model for which task |
| Fallback Chains | Graceful degradation |
| Token Budgets | Cost management strategies |

**Deliverable**: Multi-model router with automatic fallback

---

### Module 04: Memory & RAG (Weeks 10-12)
**Theme**: Persistent systems, RAG, vector databases

| Topic | Description |
|-------|-------------|
| Memory Types | Short-term vs long-term |
| Embeddings | Vector representations |
| RAG Patterns | Retrieval-augmented generation |

**Deliverable**: Personal knowledge base with semantic search

---

### Module 05: Agents (Weeks 13-15)
**Theme**: Multi-agent architectures, tool use

| Topic | Description |
|-------|-------------|
| Agent Patterns | Specialized AI microservices |
| State Machines | Workflow orchestration |
| Tool Integration | External API and code execution |

**Deliverable**: Multi-agent system with tool orchestration

---

### Module 06: Capstone (Weeks 16-24)
**Theme**: Build your personal AI agent platform

| Topic | Description |
|-------|-------------|
| Architecture | Modular system design |
| Deployment | Production infrastructure |
| Portfolio | Professional presentation |

**Deliverable**: Personal AI Agent Platform (the course's ultimate goal)
