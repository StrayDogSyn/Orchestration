# Bootcamp Modules

This directory contains the six core modules of the AI Orchestrator Bootcamp. The curriculum is designed to be self-paced and takes 17-24 weeks to complete.

## Module Structure

Each module contains:
- **lectures/**: Instructional content and presentations
- **labs/**: Hands-on exercises and practice problems
- **projects/**: Portfolio-building projects
- **resources/**: Additional reading materials and references

## Curriculum Overview

### Module 01: Foundations (Weeks 1-3)
Introduction to programming fundamentals, AI concepts, and development environment setup.
- Python fundamentals
- Version control with Git
- Introduction to AI and LLMs
- Development environment setup

### Module 02: AI APIs (Weeks 4-7)
Working with AI APIs from Anthropic, OpenAI, and Google.
- Understanding API authentication
- Prompt engineering basics
- API integration patterns
- Free-tier optimization

### Module 03: Data Processing (Weeks 8-11)
Data handling, embeddings, and vector databases.
- Data preprocessing and cleaning
- Working with embeddings
- ChromaDB and vector search
- Building knowledge bases

### Module 04: Application Development (Weeks 12-16)
Building full-stack applications with AI capabilities.
- Web application frameworks
- Frontend integration
- Backend services
- User experience design

### Module 05: Deployment (Weeks 17-20)
Deploying and maintaining AI-powered applications.
- Cloud platforms (free-tier)
- CI/CD pipelines
- Monitoring and debugging
- Cost optimization

### Module 06: Capstone (Weeks 21-24)
Final portfolio project demonstrating all learned skills.
- Project planning and scoping
- Implementation
- Documentation
- Presentation and portfolio development

## Assessment Approach

All modules use portfolio-based assessment:
- No traditional exams
- Focus on building demonstrable skills
- Real-world project work
- Emphasis on accessibility and justice-focused outcomes
