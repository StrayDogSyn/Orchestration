# Module 02: AI APIs

**Duration**: Weeks 4-7 (approximately 4 weeks)

## Overview

This module teaches students how to integrate with major AI APIs including Anthropic Claude, OpenAI GPT, and Google Gemini. Focus is on free-tier usage and effective prompt engineering.

## Learning Objectives

By the end of this module, students will be able to:
- Authenticate and call AI APIs
- Write effective prompts for different use cases
- Handle API responses and errors
- Optimize for free-tier limitations
- Compare different AI models

## Module Structure

### Lectures
Core instructional content covering:
- API fundamentals (REST, authentication, rate limits)
- Anthropic Claude API
- OpenAI API
- Google Generative AI (Gemini)
- Prompt engineering techniques
- Free-tier optimization strategies

### Labs
Hands-on exercises including:
- API authentication setup
- Basic API calls and responses
- Prompt experimentation
- Error handling practice
- Model comparison exercises

### Projects
Portfolio-building activities:
- Multi-model chatbot
- Prompt engineering showcase
- AI-powered tool or utility

### Resources
- Official API documentation
- Prompt engineering guides
- Free-tier comparison charts
- Example prompts and use cases

## Assessment

Portfolio components:
- Working application using at least two AI APIs
- Documented prompt engineering process
- Cost and performance analysis write-up
