# Module 06: Capstone

**Duration**: Weeks 21-24 (approximately 4 weeks)

## Overview

The capstone module is where students bring together all their learning to create a comprehensive portfolio project. This project should demonstrate mastery of AI-augmented development and showcase their unique skills and interests.

## Learning Objectives

By the end of this module, students will be able to:
- Plan and scope a complete AI application
- Implement a full-stack solution from scratch
- Create comprehensive documentation
- Present technical work effectively
- Build a professional portfolio

## Module Structure

### Lectures
Guidance on:
- Project planning and scoping
- Portfolio best practices
- Technical presentation skills
- Career development in AI/tech
- Justice-focused technology applications
- Community building and networking

### Labs
Support activities including:
- Project proposal reviews
- Code reviews and feedback sessions
- Documentation workshops
- Presentation practice

### Projects
**The Capstone Project**:
Students create a significant AI-powered application that:
- Solves a real-world problem
- Demonstrates technical skills from all modules
- Showcases creativity and innovation
- Includes comprehensive documentation
- Is deployed and publicly accessible

### Resources
- Project planning templates
- Documentation guides
- Portfolio website templates
- Presentation resources

## Assessment

Portfolio components:
- Complete, deployed application
- Source code repository
- Technical documentation
- User documentation/README
- Project presentation/demo
- Reflection on learning journey

## Project Ideas

Consider projects that:
- Address justice or accessibility issues
- Help underserved communities
- Improve education or learning
- Enhance creativity or productivity
- Demonstrate social impact

Examples:
- Job search assistant for justice-impacted individuals
- Accessible educational content generator
- Community resource finder
- Personal legal document helper
- Skills assessment and career guidance tool
