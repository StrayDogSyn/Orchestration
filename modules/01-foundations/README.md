# Module 01: Foundations

**Duration**: Weeks 1-3 (approximately 3 weeks)

## Overview

This foundational module introduces students to programming fundamentals, AI concepts, and development environment setup. Students will build a strong base for AI-augmented development.

## Learning Objectives

By the end of this module, students will be able to:
- Write Python programs with proper syntax and style
- Use Git for version control
- Understand basic AI and LLM concepts
- Set up a professional development environment
- Navigate the command line interface

## Module Structure

### Lectures
Core instructional content covering:
- Python basics (variables, data types, control flow)
- Functions and modules
- Git and GitHub fundamentals
- Introduction to AI/LLMs
- Development tools and IDEs

### Labs
Hands-on exercises including:
- Python programming challenges
- Git workflow practice
- Environment setup exercises
- Code organization and structure

### Projects
Portfolio-building activities:
- Personal GitHub profile setup
- First Python application
- Documentation portfolio piece

### Resources
- Python documentation and tutorials
- Git cheat sheets
- AI/LLM introductory articles
- Community support links

## Assessment

Portfolio components:
- Functional GitHub profile with bio and repositories
- Working Python scripts demonstrating core concepts
- Documentation of learning journey
