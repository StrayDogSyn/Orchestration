# Pedagogy

This directory contains educational philosophy, teaching methodologies, and learning strategies for the AI Orchestrator Bootcamp.

## Core Principles

### Portfolio-Based Assessment
Students demonstrate learning through:
- Real-world projects
- Public repositories
- Documentation and reflection
- Peer review participation
- No traditional exams or grades

### Justice-Focused Accessibility
Ensuring equitable access for all learners:
- Free-tier tools and resources
- Self-paced learning model
- Multiple learning modalities
- Support for diverse backgrounds
- Removal of traditional barriers

### Fundamentals-First Approach
Building strong foundations:
- Core programming skills
- Computer science fundamentals
- Problem-solving strategies
- Critical thinking development
- Professional practices

### AI-Augmented Learning
Teaching students to:
- Use AI as a learning companion
- Develop effective prompting skills
- Maintain critical thinking
- Understand AI limitations
- Build with AI responsibly

## Teaching Methods

- Project-based learning
- Peer collaboration
- Self-directed exploration
- Hands-on labs and exercises
- Real-world problem solving
- Continuous feedback loops

## Support Structure

- Self-paced curriculum design
- Clear learning objectives
- Scaffolded complexity
- Multiple examples and resources
- Community support channels
