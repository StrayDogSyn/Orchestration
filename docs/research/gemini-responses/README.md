## <img src="https://cdn.jsdelivr.net/npm/@tabler/icons@latest/icons/pencil.svg" width="20" style="filter: invert(28%) sepia(23%) saturate(1377%) hue-rotate(94deg) brightness(96%) contrast(88%);" /> How to Use

1. **Run Gemini prompts** from [`../RESEARCH_PROMPTS.md`](../RESEARCH_PROMPTS.md)
2. **Save responses here** with naming format: `0X-[topic]-YYYYMMDD.md`
3. **Extract key findings** for synthesis
4. **Keep originals local**, commit only synthesized insights
