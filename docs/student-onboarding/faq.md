### Q: My API key isn't working / I got a 429 Error.
**A:** You hit the rate limit.
1.  Check if you are sending requests too fast. Implement the "Exponential Backoff" code from Module 1, Lecture 6.
2.  Check if your free trial expired.
3.  Switch to a different provider (e.g., from Gemini to Claude) as a fallback.

### Q: Can I use ChatGPT (OpenAI)?
**A:** Yes, but be careful.
*   **GPT-4o-mini** has a free tier/cheap API.
*   **GPT-4** is paid ($20/mo).
*   **Constraint:** You must ensure your project *can* run on free tools. If you build something that *only* works on GPT-4, you failed the Zero-Cost Mandate.

### Q: I'm on an old laptop. Can I do this?
**A:** Yes!
*   We use cloud APIs (Gemini/Claude), so the "heavy lifting" happens on Google/Anthropic's servers, not your laptop.
*   If VS Code is too slow, use **Google Colab** (runs in your browser).
