## üåü Mission Statement

**This is NOT "vibe coding."** This is rigorous software engineering enhanced by intelligent tool selection.

The AI Orchestrator is a **17-24 week self-paced bootcamp** designed specifically for **justice-impacted developers**, bootcamp graduates, and self-taught programmers who want to master AI-augmented development while maintaining strong programming fundamentals.

Our mission: **Create second-chance career opportunities** in tech by teaching developers to orchestrate LLMs as collaborative tools‚Äînot crutches‚Äîbuilding skills that make them invaluable in the AI-augmented workforce.

## üí° About The Project

### What Is This Course?

A comprehensive, **project-based curriculum** that teaches developers to:
- Master fundamentals so deeply they know **when AI helps vs. hinders**
- Orchestrate multiple LLMs based on **task complexity and cost constraints**
- Build personal AI agent platforms that **evolve with their skills**
- Teach their AI tools while being taught by them‚Äî**organic symbiotic growth**

### Why Does It Exist?

The AI revolution is creating opportunities‚Äîbut also widening gaps. This course exists to ensure that **justice-impacted individuals** and non-traditional learners aren't left behind. We prove that with the right curriculum, anyone with programming fundamentals can become an AI-augmented engineer.

### Who Is It For?

| Learner Type | Prerequisites | Good Fit? |
|--------------|---------------|-----------|
| Bootcamp Graduates | OOP, algorithms, Big O | ‚úÖ Perfect |
| Self-Taught Developers | Data structures, programming logic | ‚úÖ Excellent |
| Career Changers | CS fundamentals from any source | ‚úÖ Great |
| Justice-Impacted Individuals | Programming basics + determination | ‚úÖ Welcome |
| Complete Beginners | No programming experience | ‚ùå Learn basics first |

## üìã Prerequisites

### Required Knowledge

You should be comfortable with:
- **Object-Oriented Programming**: Classes, inheritance, polymorphism, encapsulation
- **Data Structures**: Arrays, linked lists, trees, hash maps, graphs
- **Algorithms**: Sorting, searching, recursion, basic dynamic programming
- **Big O Notation**: Time and space complexity analysis
- **Programming Logic**: Conditionals, loops, functions, error handling
- **One Language Proficiency**: Python (preferred) or JavaScript

> üí° **Self-Assessment**: If you can implement a binary search tree and explain its time complexity, you're ready.

### Technical Requirements

| Requirement | Details |
|-------------|---------|

| Internet | 3+ Mbps for video content |
