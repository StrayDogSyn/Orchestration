# Assets

This directory contains visual and media assets for the AI Orchestrator Bootcamp.

## Contents

This directory will contain:
- Logos and branding
- Diagrams and illustrations
- Presentation slides
- Screenshot examples
- Icons and graphics
- Marketing materials

## File Organization

Assets are organized by:
- Type (images, videos, documents)
- Module or topic
- Purpose (documentation, presentation, marketing)

## Usage Guidelines

- Use accessible alt text for images
- Optimize file sizes
- Use open formats when possible
- Include source files when available
- Maintain proper licensing information
