# Resources

This directory contains supplementary learning materials for the AI Orchestrator Bootcamp.

## Structure

### Videos (`videos/`)
Video content including:
- Lecture recordings
- Tutorial videos
- Guest speaker presentations
- Project walkthroughs
- Student success stories

### Articles (`articles/`)
Written content including:
- Technical blog posts
- Industry insights
- Best practices guides
- Tool comparisons
- Case studies

### Templates (`templates/`)
Reusable templates including:
- Project scaffolding
- Documentation templates
- Code style guides
- Portfolio website templates
- Presentation templates

## Contributing

Students and instructors can contribute resources following these guidelines:
- Ensure content is accessible and inclusive
- Use open licenses when possible
- Prioritize free-tier and open-source tools
- Include proper citations and attributions
- Test all code examples and tutorials

## Resource Organization

Resources are tagged and categorized by:
- Module relevance
- Skill level (beginner, intermediate, advanced)
- Content type (video, article, tutorial, reference)
- Tool or technology focus
